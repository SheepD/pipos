# piPOS

A basic Point of Sales System built for the raspberry-pi

# Features

## User Authentication

## Inventory Searching

# System Requirements

1. Node version 12.13.0

# Installation Guide

1. `git clone git@github.com:SheepD/pipos.git`
1. `cd pipos`
1. `npm install`
1. `npm run serve`
1. `npm run start`
